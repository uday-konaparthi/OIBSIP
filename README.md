# OIBSIP — Web Development & Design Internship

Tasks completed for the **Oasis Infobyte Summer Internship Program (OIB-SIP)**, Web Development & Design track — **Level 1**.

Built with **React**, **Vite**, and **Tailwind CSS**.

## 📁 Projects

### 1. Landing Page — `LANDING_PAGE/`
A product landing page for **SyncSpace**, a concept real-time collaborative code editor. Features an animated multi-cursor code editor mockup as the hero, a features section, and a "how it works" walkthrough.

**Highlights:** responsive layout, hover/scroll micro-interactions, dark developer-tool aesthetic.

### 2. Portfolio — `PORTFOLIO/`
A personal developer portfolio showcasing skills, projects, work experience, and certifications.

**Sections:** About, Tech Stack, Projects, Experience, Certifications.

### 3. Temperature Converter — `TEMP_CONVERTER/`
A functional Celsius / Fahrenheit / Kelvin converter styled like a physical measuring-instrument panel, with an LED-style digital readout.

**Highlights:** live conversion on input, switchable input unit, instant dual-unit output.

## 🛠️ Tech Stack

- React (Vite)
- Tailwind CSS
- Deployed on Vercel

## 🚀 Running Locally

Each project is a standalone Vite app. To run any one of them:

```bash
cd LANDING_PAGE      # or PORTFOLIO / TEMP_CONVERTER
npm install
npm run dev
```

Then open the local URL Vite prints (typically `http://localhost:5173`).
